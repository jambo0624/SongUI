
# 开始使用
请先确保[安装](#/doc/install)了本组件库

然后引入要使用的组件

```js
import { Button, Tabs, Switch, Dialog } from 'song-ui' 
```

下面是代码示例:
```vue
<template>
  <div>
  <Button>按钮</Button>
</div>
</template>
<script >
  import { Button } from 'song-ui' 
  export default {
    components: { Button }
  }
</script>
```
